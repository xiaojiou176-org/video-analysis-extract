# Code of Conduct

## Scope

This repository is a limited-maintenance, source-first engineering project.
Respectful, concise, and evidence-based collaboration is required in issues,
pull requests, security reports, and code review.

## Expected Behavior

- stay factual and reproducible
- separate repo-side facts from external/platform blockers
- avoid posting secrets, private routes, or sensitive operational details
- keep disagreement technical, specific, and respectful

## Unacceptable Behavior

- harassment, threats, or personal attacks
- posting exploit details in public issues when a private security route exists
- demanding hosted-service support or private-environment debugging not promised by this repository
- misrepresenting historical artifacts as current proof

## Reporting

For non-sensitive conduct problems, use the normal issue flow.
For sensitive security-adjacent content, use the private security route described in `SECURITY.md`.
